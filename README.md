# Stretch Robot

Stretch robot control code

## Getting Started

1. Create a Conda environment:

```bash
conda create --name stretch python=3.10
```

2. Install dependencies:

```bash
make install-dependencies
```

3. Install the package:

```bash
make install
```

4. Run the bot:

```bash
bot
```
